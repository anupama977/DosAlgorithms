This repository presents the implementation of algorithms related to the research topic **"Design of Intelligent Resilient Control Against Adversarial Attacks in Networked Control Systems."**

*"improved_ada_threshold.py" computes the optimal event-trigger threshold (`sigma_max`).
* "performance_controller.py" uses the computed threshold to determine the maximum allowable delay (`tau_max`) and the controller gain matrix (`K`).

These algorithms contribute to the design and evaluation of resilient networked control systems under adversarial attack scenarios.
